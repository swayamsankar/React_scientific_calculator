# 🧮 React Scientific Calculator

## 📌 Overview

The **React Scientific Calculator** is a modern and responsive web-based calculator built using React. It performs basic as well as scientific calculations with a clean and user-friendly interface.

---

## 🚀 Features

* ➕ Basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
* 📐 Scientific functions (Trigonometry, Log, Exponents, etc.)
* 🎨 Clean and responsive UI
* ⚡ Fast and interactive performance
* 🔒 Safe evaluation (no use of `eval`)

---

## 🛠️ Tech Stack

* **React.js**
* **JavaScript (ES6+)**
* **HTML5**
* **CSS3**

## ▶️ How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/React_scientific_calculator.git
cd React_scientific_calculator
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm start
```

4. Open in browser:

```
http://localhost:3000
```
